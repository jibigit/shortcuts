Cheaalt CtrlSheet ShortcuMajts

Cette page fournit un aperçu des raccourcis clavier qui peuvent vous aider à utiliser votre bureau et les applications plus efficacement. Si vous ne pouvez pas utiliser de souris ni d'appareil de pointage, consultez Navigation au clavier pour savoir comment naviguer dans les interfaces uniquement via le clavier.

CtrlAltt Ouvrir un terminal
CtrlMajc Dans un terminal Copier le texte
CtrlMajv Dans un terminal Coller le texte
AltTabv Changer rapidement de fenêtre
Altf4 Fermer la fenêtre actuelle
Alt f10 Maximiser une fenêtre
AltImpr.Ecrani Tue tous les programmes
MajSuppr Supprimer un fichier ou un répertoire sans passer par la corbeille (suppression définitive)
AltF1 Menu applications
CtrlAltl Verrouiller l'écran
Altf7 Déplacer la fenêtre
CtrlAltd Cacher ou voir toutes les fenêtres du bureau
AltImpr.Ecrano Quit the app.
